# EvolutionV2

Monitoring Species Evolution under different terrains

# Configuration

|   Option   | Values |              Description              |
| :--------: | :----: | :-----------------------------------: |
| DEBUG_MODE | ON/OFF | Debug Mode. Press V Key to see Vision |
| UNIT_TEST  | ON/OFF |               Unit Test               |

# Compilation

```bash
mkdir build
cd build
cmake ../
make -j32
```

# Run

./Evolution
